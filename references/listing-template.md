# 咸鱼出售文案模板

## 标题公式

```
[{机型}] {CPU亮点} + {核心卖点} + {出售范围}

示例：
【ITX小主机】i5-8400H 4核8线程 + 500G固态 + 整套到手即用
【台式机】i5-8400 6核 + 8G + 256G固态 + 独显 办公游戏两用
【办公电脑】i3-8100 8G 固态 含显示器键鼠 整套打包
```

## 模板A：整套打包出售（含显示器+键鼠）

```
【{机型描述}】{CPU型号} + {内存}+ {硬盘}+ 显示器键鼠整套

 配置：
 CPU：{完整CPU名称+核心线程数}
 主板：{主板型号}
 内存：{容量} {类型} {频率}
 硬盘：{品牌} {容量} {类型}
 显卡：{显卡型号}（亮机卡/核显/游戏卡）
 显示器：{尺寸} {分辨率}
 键盘鼠标：{描述}
 系统：{操作系统版本}（已激活，到手即用）

 适合：{目标用户描述}
 优势：{3-5个卖点}

 说明：
1. {特殊说明1：魔改/ES等}
2. {特殊说明2：显卡限制}
3. {特殊说明3：其他注意事项}

 整套打包价：{价格}元（含主机+显示器+键鼠）
同城自提优先，支持现场验机。
诚心出，屠龙刀勿扰。有问题随时问，知无不言。
```

## 关键词标签建议

发帖时在末尾添加这些标签增加搜索曝光：

```
#二手电脑 #台式机 #{CPU代次}代i5 #{城市}二手 #{品牌} #{ITX/小主机/台式} #办公电脑 #学生电脑 #固态硬盘
```

## 魔改配置声明模板

如果检测到魔改配置（笔记本CPU上台式、魔改BIOS等），必须添加以下声明：

```
 关于魔改说明：
本机采用{主板型号}通过dsanke魔改BIOS支持{CPU型号}（8代笔记本CPU）。
已稳定使用{时间}，日常办公/影音无任何问题。
魔改配置性价比高，但不适合追求原厂配置的强迫症买家。
介意勿拍，懂行的来。
```

## 照片拍摄清单

在文案中建议用户拍摄以下照片：

1. 主机正面全景（侧板关闭）
2. 主机内部（打开侧板，展示CPU散热、内存、显卡）
3. 主机背面接口
4. 显示器亮屏展示（显示桌面或鲁大师配置页）
5. 键鼠特写
6. 全家福（所有出售物品摆在一起）
7. 鲁大师/CPU-Z 配置截图（如有）
