## 整机估价速算公式

```
散件总价上限 = SUM(各部件估价上限)
散件总价下限 = SUM(各部件估价下限)

整机打包价 × 0.85 = 散件总价（打包应有折扣）
快速出手价 = 整机打包价 × 0.85
建议定价 = 整机打包价 × 0.95
理想售价 = 整机打包价 × 1.05
```

## 咸鱼定价心理学

- 标价 **XX9** 或 **XX8** 比整数更有吸引力（如 599 vs 600）
- 挂价比心理底价高 **100元**，给买家砍价空间
- "整套打包"比拆件卖总价低10-15%，但省心
- 同城自提 + 面交可溢价5-10%（免邮费+信任感）
